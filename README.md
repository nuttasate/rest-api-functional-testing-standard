# REST API Functional Testing Standard

This repository contains a small React app that renders the REST API functional and security testing checklist.

## Run locally

```bash
npm install
npm run dev
```

## Deploy on GitHub Pages

1. Push this repository to GitHub.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **GitHub Actions**.
4. Push to the `main` branch or run the `Deploy GitHub Pages` workflow manually.

The workflow builds the app and publishes the `dist` folder to GitHub Pages automatically.
